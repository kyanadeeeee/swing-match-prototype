ゴルフスイング分析アプリ「**Swing Match**」のプロトタイプです。AI技術でスイングを詳細分析し、**4つの異なる観点**から最適なクラブとシャフトの組み合わせを提案します。

## ✨ 最新機能ハイライト

### 🎯 **革新的な4観点クラブ推奨システム**
従来の「1つの最適解」から、ユーザーの目標に応じた**4つの戦略的選択肢**を提供：

- **🚀 飛距離重視** - より遠くへ飛ばしたい方に（+15-20ヤード向上）
- **🎯 正確性重視** - フェアウェイキープ率を上げたい方に（+25%向上）  
- **🛡️ やさしさ重視** - ミスヒットに強いクラブをお探しの方に（-30%飛距離ロス軽減）
- **🎮 コントロール重視** - 球筋を自在に操りたい上級者向け（-40%左右ブレ改善）

### 🧭 **先進的なナビゲーションUX**
- **プログレスインジケーター** - 全体のフローが一目で分かる5ステップ表示
- **ブレッドクラムナビゲーション** - 現在位置の明確化と簡単な戻り操作
- **フローティングアクションボタン** - 次にすべきアクションが常に明確

## 🌟 主要機能

### 📊 **高度なAIスイング分析**
- **詳細メトリクス**: ヘッドスピード、テンポ、スイング軌道、タイミング分析
- **多角的可視化**: 線グラフ、棒グラフ、レーダーチャートによる包括的表示

### 🏌️ **リアルタイムバーチャル試打**
- **インタラクティブシミュレーション**: 選択したクラブでの弾道アニメーション

- **カスタマイズ**: 豊富なクラブ・シャフト組み合わせから選択

### 🚚 **統合レンタルシステム**
- **シームレス連携**: 分析→試打→レンタルの一貫したフロー
- **詳細料金計算**: 期間別料金、配送料、税込み価格の自動計算
- **完了通知**: 申し込み完了時の美しいアニメーション

### 📱 **モバイルファーストUX**
- **レスポンシブデザイン**: スマートフォンからデスクトップまで最適化
- **タッチ操作対応**: スマートデバイスでの直感的な操作
- **動的ページタイトル**: ブラウザタブでの分かりやすい表示

## 🎮 使用方法

### 1️⃣ **クイックデモ**
動画ファイルなしでも「**🧪 テスト用サンプルで分析**」ボタンで全機能を即座に体験可能

### 2️⃣ **フルフロー体験**
1. **ホーム画面** → 「📹 スイングを分析する」
2. **動画撮影** → テスト分析または実際の動画アップロード  
3. **分析結果** → 4つの推奨から目標に合ったクラブを選択
4. **バーチャル試打** → 選択したクラブでシミュレーション
5. **レンタル申し込み** → 詳細情報入力で完了

## 🎨 デザインシステム

### **カラーパレット**
```css
Primary: #2E7D32 (ダークグリーン)
Primary Light: #4CAF50 (メイングリーン)  
Secondary: #81C784 (ライトグリーン)
Accent: #A5D6A7 (極薄グリーン)
Background: #F1F8E9 (背景グリーン)
```

### **タイポグラフィ**
- **フォントファミリー**: Inter + システムフォント
- **スケール**: 12px〜32px の段階的サイズ設定
- **ウェイト**: Light(300) 〜 Bold(700) の5段階

### **スペーシング・アニメーション**
- **統一スペーシング**: 4px基準の8段階システム
- **トランジション**: 0.15s〜0.5s の3段階スピード
- **イージング**: ease-in-out による自然な動作

### **強化されたカードデザイン**
- **ガラスモーフィズム効果**: `backdrop-filter: blur(10px)` による現代的な質感
- **グラデーション背景**: `linear-gradient(145deg, #ffffff 0%, #f8fffe 100%)` で立体感を演出
- **多層シャドウ**: outer shadow + inset highlight で自然な奥行き
- **レスポンシブアニメーション**: ホバー時の `translateY` と `box-shadow` 変化
- **アクセントボーダー**: 上部のカラフルなグラデーションライン
- **選択状態の視覚化**: 推奨カードの半透明オーバーレイ効果
